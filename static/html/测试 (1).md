## 定理 2.3.1

**$x_n \neq 0$，则 $\{x_n\}$ 是无穷大量的充要条件是 $\left\{\frac{1}{x_n}\right\}$ 是无穷小量。**

---

### 证：

#### 必要性（$\Rightarrow$）：假设 $\{x_n\}$ 是无穷大量，证 $\left\{\frac{1}{x_n}\right\}$ 是无穷小量

$$\forall \varepsilon > 0, \quad \text{令 } G = \frac{1}{\varepsilon} > 0$$

$$\exists N, \quad \forall n > N, \quad |x_n| > G = \frac{1}{\varepsilon}$$

$$\Rightarrow \left|\frac{1}{x_n}\right| < \varepsilon$$

---

#### 充分性（$\Leftarrow$）：假设 $\left\{\frac{1}{x_n}\right\}$ 是无穷小量，证 $\{x_n\}$ 是无穷大量

$$\forall G > 0, \quad \text{令 } \varepsilon = \frac{1}{G} > 0$$

$$\exists N, \quad \forall n > N, \quad \left|\frac{1}{x_n}\right| < \varepsilon = \frac{1}{G}$$

$$\Rightarrow |x_n| > G$$
